## 1. NPM

```shell
# 单次使用淘宝镜像
npm [命令] --registry=https://registry.npm.taobao.org

# 默认设置淘宝镜像 npm config
npm config set registry https://registry.npm.taobao.org

# 设置环境变量 npm set
npm set init-author-name 'CregskiN'

# 查看某个包的信息 npm info
npm info react

# 查找npm仓库 npm search
npm search react

# 树状展示当前项目安装的模块 npm list
npm list --g lobal
```
